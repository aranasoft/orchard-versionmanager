Name: Version Manager
AntiForgery: enabled
Author: Renaud Paquay
Website: http://orchardproject.net
Version: 0.9.0
OrchardVersion: 0.9.0
Description: Provide admin UI to display the revision history of content items, as well as a recycle bin to restore deleted content items. The list of versions can be accessed through a new "Versions" link in the "Content Items" admin page. The "Recycle Bin" menu is displayed under the "Content" admin menu item.
